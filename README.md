# ☕ Coffee Haven - Aesthetic Coffee Landing Page

A modern, elegant, and fully responsive coffee shop landing page built using **HTML, CSS, and JavaScript**. The website delivers a premium café experience with smooth scrolling, stylish animations, and a clean user interface.

## 🌐 Live Demo

**🔗 Website:** https://caffe-web.netlify.app/

---

## ✨ Features

* 🎨 Modern & Aesthetic UI Design
* ☕ Premium Coffee Shop Landing Page
* 📱 Fully Responsive Design
* 🚀 Smooth Scroll Experience
* ✨ Elegant Hover Animations
* 🖼️ Beautiful Hero Section
* 📋 Interactive Navigation Bar
* 🍩 Featured Coffee & Menu Sections
* 💬 Customer Testimonials
* 📩 Newsletter Subscription Section
* 📍 Contact & Footer Section
* ⚡ Fast Loading Performance
* 🌙 Luxury Dark Theme


```bash

```
